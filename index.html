PROJECTS

Traffic on Texas Highways

Big data project analyzing real-time traffic using streaming pipelines and cloud analytics.

This project focused on analyzing Texas highway traffic patterns by integrating real-time data streams using REST APIs and Apache Kafka. The datasets were cleaned and refined using OpenRefine, significantly improving data reliability. Leveraging Apache Spark and Hadoop on Google Cloud Platform, the project assessed the impact of weather on traffic flow and enhanced model efficiency by 25%. SQL queries and data pipelines were optimized to reduce processing time by 20%. The insights were visualized through interactive Tableau dashboards, enabling transportation authorities to make informed decisions and reduce peak-hour congestion by 15%.

Crime Chronicles: Tracing Trends in USA Crime

Interactive data storytelling with Tableau to uncover U.S. crime trends and regional patterns.

This project explored crime trends across the United States using an interactive Tableau dashboard. By integrating charts, KPIs, and custom filters, it enabled users to navigate crime data by state, year, and category. Data cleaning was performed in Tableau Prep, involving format normalization, removal of duplicates, and grouping of categorical fields, which improved data consistency by 30%. The dashboard utilized Tableau’s geospatial mapping tools to visualize crime hotspots, weapon usage, and regional death rates, providing actionable insights for both policy formulation and law enforcement strategies.

Drug-Target Interaction Prediction

Deep learning and cloud-based pipeline for scalable drug discovery analytics.

This project aimed to accelerate drug discovery by predicting drug-target interactions using deep learning models built in PyTorch, achieving a 20% boost in predictive accuracy. Large-scale genomic and proteomic datasets were processed using Databricks to identify promising drug candidates. An end-to-end data pipeline was developed using Azure Data Factory and Snowflake for efficient data ingestion and storage. The model was deployed using Docker containers on Azure Cloud, reducing inference time by 30%. The solution also featured automated monitoring and logging for reproducibility, supporting its integration into clinical research environments.

US Healthy Life Supplements 2024

AI-driven forecasting and sentiment analysis for sales optimization in the health sector.

This project focused on optimizing sales and customer engagement for a supplement company using machine learning and NLP techniques. Models for sales forecasting and pricing prediction were developed with 90% accuracy and an R² score of 0.99. Extensive data cleaning and exploratory data analysis (EDA) improved data reliability by 95%, while advanced feature engineering further enhanced prediction performance by 20%. Natural language processing (NLP) models were implemented for sentiment analysis and named entity recognition, reducing false positives by 25%. The entire solution was deployed on AWS SageMaker with 99.9% uptime and paired with a Power BI dashboard, leading to a 15% increase in customer satisfaction and improved inventory management.

Coronary Artery Disease Prediction

Hybrid ensemble models for early CAD diagnosis using real-world clinical data.

This project developed a predictive system for early detection of coronary artery disease using hybrid machine learning models. Multiple datasets from Mendeley, Kaggle, and UCI were consolidated, totaling 1,738 records with 12 clinical features. After rigorous data cleaning, hypothesis testing, and exploratory data analysis, advanced feature engineering and scaling were applied. Baseline performance was established with individual classifiers, followed by the implementation of ensemble techniques such as stacking and soft voting using models like Random Forest, XGBoost, LightGBM, and SVC. The hybrid models delivered superior results across metrics including accuracy, precision, recall, F1-score, and AUC-ROC, demonstrating strong potential for real-world healthcare applications.
